Invokes a callback at an interval until instructed to stop.

This type of task can be used to perform open-ended or non-deterministic actions. It will run until instructed to complete (or error) by the provided callback.

Note that you are recommended to use task for animations and DOM manipulation.

Constructor tr.TimerTick(callback, interval, name)

Instance methods

complete(data)

Call this method to mark the task as complete.

error(data, errorMessage)

Call this method to mark the task as errored.

setInterval(interval)

Adjust the interval between timer ticks.
