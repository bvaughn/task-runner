Lightweight interface to create a dependency-graph task.

Constructor tr.Chain(completedCallback, erroredCallback)

Instance Methods

else(...tasks)

Alias for or method.

first(...tasks)

Add one or more tasks to the beginning of the chain.

or(...tasks)

Add one or more tasks to be run only if one of the previously-added tasks fail.

Tasks passed to this method will be run in parallel. To run fallback tasks in serial, chain multiple calls to or().

otherwise(...tasks)

Alias for or method.

then(...tasks)

Add one or more tasks to be run after the tasks already in this chain have been run.
