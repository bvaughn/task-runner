Runs a series of tasks until one of them successfully completes.

This type of task completes successfully if at least one of its children complete. If all of its children error, this task will also error.

This type of task has very limited use cases. Consider tr.Chain, tr.Composite or tr.Graph before proceeding.

Constructor tr.StopOnSuccess(tasks, taskName)

Instance Methods

add(task)

Adds a task to the list of child tasks.

This method should only be called before the task is run. Adding child tasks while running is not a supported operation.

addAll(tasks)

Adds a set of tasks to the list of children.

This method should only be called before the task is run. Adding child tasks while running is not a supported operation.

remove(task)

Removes a task from the list of child tasks.
