Task Runner

Task Runner is a collection of low-level libraries designed to make JavaScript application development easier.

Download Task Runner Learn More

Forever free, open source, and easy to use.

Task Runner is built with the Closure Compiler, but it has no runtime dependencies on closure libraries. It can be used with any JavaScript framework- Angular, React, jQuery, etc.

If your project uses the Closure library, use the Task Runner source so that Closure can compile it along with the rest of your application.
