Why another async framework? Why not just use promises?

There are many async JavaScript frameworks but Task Runner offers a few advantages:
  • It focuses on interruptability (pause and resume) which makes it particularly useful for things like animations.
  • It provides robust ways to compose tasks (treating groups of tasks the same way you would treat a single task).
  • Tasks can be converted to and from Promises, making it easy to use integrate with other libraries.

Will it work with other frameworks?

Yes! Task Runner does not depend on any other libraries. It's small (7.5k compressed) and can be used with any JavaScript framework- Angular, React, jQuery, etc.
It's also built with TypeScript- so if your project uses TypeScript, you'll have an even easier integration.
