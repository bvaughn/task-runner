Task Runner

Task Runner is a collection of low-level libraries designed to make JavaScript application development easier.

With just a few characters you can chain together asynchronous operations.

But that's only the beginning. Keep reading to learn more...

View Source Get Started

Works with your favorite frameworks!

Task Runner has no runtime dependencies on any external libraries. It can be used with any JavaScript framework- Angular, React, jQuery, etc.

It's also built with TypeScript- so if your project uses TypeScript, you'll have an even easier integration.
