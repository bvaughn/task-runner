Creating Tasks

Task Runner includes several reusable tasks but you can also create your own. There are 2 basic approaches:

Creating Tasks via Inheritance

Extend tr.Abstract task and override the runImpl, interruptImpl, and resetImpl methods. Learn more...

Creating Tasks via Composition

Use the built in tr.Decorator task to decorate objets and automatically turn them into fully functioning tasks. Learn more...

Creating Tasks with Closures

For simple operations, you can use the built in tr.Closure task to decorate functions turn it into a task. Learn more...
