Creating Tasks

Task Runner includes several reusable tasks but you can also create your own. There are 2 basic approaches:

Creating Tasks via Inheritance

Extend task and override the runImpl, interruptImpl, and resetImpl methods.

Creating Tasks via Composition

Use the built in task to decorate objets and turn them into fully-functioning tasks.

Creating Tasks with Closures

For simple operations, you can use the built in task to decorate functions and automatically turn them into tasks.
