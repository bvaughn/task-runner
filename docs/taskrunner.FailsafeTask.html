Class: FailsafeTask

FailsafeTask

new taskrunner.FailsafeTask(decoratedTask)

failsafe-task.js, line 22
Decorates a task and re-dispatches errors as successful completions. This can be used to decorate tasks that should be run at a particular time, but are not essential.
Name Type Description
decoratedTask taskrunner.Task Decorated task to be run when this task is run.
Example
// Runs the decoratedTask and successfully completes as soon as it errors or completes.
var task = new taskrunner.FailsafeTask(decoratedTask);
task.run();
